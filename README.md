## Features

- **Kanban Board**: Drag and drop tasks across columns (To Do, In Progress, Done)
- **Task Messaging**: Comment and chat directly on any task
- **Team Management**: Invite new members via email and assign roles
- **Labels & Priorities**: Tag tasks with custom labels and priority levels

## Getting Started

### Prerequisites

Make sure you have the following installed:

- [Node.js](https://nodejs.org/) v18 or higher
- npm or yarn

### Installation

1. **Clone the repository**

```bash
git clone https://github.com/DashginAsgarli/projectm-dashboard.git
cd projectm-dashboard
```

2. **Install dependencies**

```bash
npm install
```


3. **Start the development server**

```bash
npm run dev
```

Visit `http://localhost:5173` in your browser.
